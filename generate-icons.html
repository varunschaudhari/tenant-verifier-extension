How to Generate Icons:

  1. Click "Generate All Icons" to create all icon sizes
  2. Right-click on each canvas and select "Save image as..."
  3. Save each icon with the correct filename (icon16.png, icon32.png, etc.)
  4. Place all saved icons in the "icons" folder of your extension
